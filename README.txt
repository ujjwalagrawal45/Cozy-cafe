# Cozy Cafe Website ☕
A static front-end cafe website built using HTML and CSS. This project simulates an online cafe ordering system where users can browse menu items, view their cart, and proceed to checkout.

## 📌 Project Overview
The Cozy Cafe website is designed to provide a modern and user-friendly interface for a cafe. It includes multiple pages such as Home, Menu, Cart, and Checkout. The project demonstrates fundamental web development concepts including layout design, responsive navigation, and form handling.

This project was created as part of a college assignment for learning front-end web development.

---

## 🧰 Technologies Used

- HTML5
- CSS3
- Google Fonts
- Font Awesome Icons
- Flexbox and Grid Layout

## 📄 Pages Description

### Home Page
Introduces the cafe and displays featured menu items and testimonials.

### Menu Page
Displays all food and beverage items categorized into sections.

### Cart Page
Shows selected items, quantity, pricing, and order summary.

### Checkout Page
Allows users to enter contact, delivery, and payment details.
